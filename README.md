# FileApiFind
文件api接口递归提取
```
python3 apifind.py -h        
usage: apifind.py [-h] [-r PATH] [-o OUTPUT]

Scan a directory.

options:
  -h, --help           show this help message and exit
  -r, --path PATH      需扫描的路径
  -o, --output OUTPUT  输出文件的名称（不含扩展名）
```
